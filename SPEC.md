# PaperHub - 学术论文搜集平台

## 1. 项目概述

- **项目名称**：PaperHub
- **项目类型**：学术论文搜集与推荐平台
- **核心功能**：聚合大模型、行为金融、巨灾保险、农业保险、普惠金融领域的论文，支持搜索、分类、订阅、收藏功能
- **目标用户**：金融学研究者、保险领域学者、AI/ML 研究人员

## 3. 功能规格

### 3.1 论文搜索与展示

- **搜索功能**
  - 支持标题、作者、关键词搜索
  - 实时搜索建议
  - 搜索历史记录

- **论文列表**
  - 卡片式展示
  - 显示：标题、作者、期刊/会议、日期、摘要(100字)、领域标签
  - 支持收藏、标记待读
  - 分页加载（每页20条）

### 3.2 领域分类体系

```
大模型 (LLM)
├── 基础模型/预训练
├── 指令微调/RLHF
├── RAG/知识增强
├── 多模态模型
└── AI Agent

行为金融
├── 投资者行为
├── 市场异象
├── 行为资产定价
└── 金融科技

巨灾保险
├── 地震保险
├── 洪水/飓风保险
├── 气候风险建模
└── 再保险

农业保险
├── 农作物保险
├── 畜牧保险
├── 天气指数保险
└── 农业信贷

普惠金融+农村金融
├── 数字普惠金融
├── 农村信贷
├── 小微金融
└── 金融排斥
```

### 3.3 用户功能

- **收藏夹**：收藏论文到个人文献库
- **阅读清单**：标记为"待读"
- **订阅**：订阅领域/关键词，新论文推送

### 3.4 数据统计（右侧栏）

- 热门论文 TOP 10
- 热门标签云
- 论文年度趋势图

## 4. 页面结构

### 4.1 首页 /dashboard
- 默认展示全部论文
- 侧边栏领域筛选
- 右侧热门内容

### 4.2 论文详情页 /paper/:id
- 完整摘要
- PDF 链接
- 相关论文推荐
- 收藏/分享按钮

### 4.3 用户中心 /profile
- 收藏列表
- 阅读历史
- 订阅管理

### 4.4 每日推送 /push
- 首页顶部横幅展示
- 个性化推荐算法
- 论文标记（New/Hot）
- 推送历史记录

---

## 5. 每日推送功能

### 5.1 推送策略
```
每日推送 = 5篇论文
├── 30% 用户订阅领域的最新论文
├── 30% 全局热门论文（高引用）
├── 20% 随机推荐（探索新领域）
└── 20% 经典必读（领域奠基之作）
```

### 5.2 论文标记
- **🆕 新论文**：30天内发布
- **🔥 热门**：引用 > 5000

### 5.3 通知方式
- Web 端推送横幅
- 邮件订阅（可选）
- 站内通知

---

## 6. 文献管理功能

### 6.1 引用导出
- **格式支持**：APA、MLA、Chicago、GB/T
- **导出格式**：BibTeX、RIS、CSV
- **一键复制**：点击复制到剪贴板

### 6.2 作者主页
- 论文数量统计
- 总引用、H指数、i10指数
- 论文列表（按时间/引用排序）
- 合作学者网络
- 研究领域分布

### 6.3 机构排名
- 按论文数量/引用排序
- 类别筛选
- 国家/地区分组
